🚀 Quick Start

  1. Click New Article in the sidebar to write a post.
  2. Fill in metadata (title, description, date, category, hero image URL).
  3. Write the article body using the formatting toolbar.
  4. Click Add to Queue — the article is added to your export queue.
  5. Go to Export & Deploy and click Download ZIP.
  6. Unzip the file and upload the blog/{slug}/ folder plus jsonassets/content.json to the repository.

📖 How to Deploy

  1. Download the ZIP file above.
  2. Unzip it — you'll see blog/ and jsonassets/ folders.
  3. Copy blog/{slug}/index.html into the same folder in the repository.
  4. Copy (replace) jsonassets/content.json in the repository.
  5. Add any hero or article images to blog/{slug}/.
  6. Commit all changes and push — the site updates automatically.
